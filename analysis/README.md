# Analysis

## MA376_Code.Rmd

This R Markdown file contains the full analysis for the Binge Drinking and Military Service study. It performs data cleaning, exploratory data analysis, logistic regression modeling, model comparison via nested ANOVA, validity condition checking, and classification rate evaluation.

## How to Run

1. Ensure R and RStudio are installed.
2. Install required packages (listed in the Rmd file's setup chunk).
3. Place the data files (`NHANES_Military_Alcohol.csv` and `ProjectData.csv`) in the `data/` directory at the project root.
4. Open `MA376_Code.Rmd` in RStudio and click **Knit**, or run chunks individually.
